# Smart Expense Tracker 💰

A web-based application to track daily expenses
anage budgets, and analyze spending habits efficiently.

## 🚀 Features
- Add, edit, and delete expenses
- Categorize expenses (Food, Travel, Shopping, etc.)
- Real-time expense tracking
- Displays total spending
- Smooth UI interactions using CSS transitions

## 🎨 UI Enhancements
- Button hover effects using CSS transitions
- Smooth animations for adding/removing elements
- Clean and responsive layout

## 🛠️ Tech Stack
- HTML
- CSS (Transitions & Styling)
- JavaScript

## 📌 What I Learned
- DOM manipulation
- Event handling
- Creating smooth UI with CSS transitions
- Managing dynamic data in frontend apps
